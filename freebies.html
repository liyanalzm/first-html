Printable Write-It-Yourself Planner

The order may look different because this template is for A5 sized paper( which is half of A4 paper). This layout is for two sided paper. Make sure you know the sequence!
(The sequence starts from number 5)
Instructions:
1. Print by following the number sequence.
2. Once printed one side, put the printed side above with the month name first to make it printing both sides.
3. Print the next number sequence
4. Once printed all, cut the a4 paper into half.
